Deploy Notes

GitHub Pages에 올릴 때

이 폴더만 배포

docs/ 안 파일은 정적 자산만 사용합니다. Flask 서버, DB, 환경변수, 민감 데이터와 분리되어 있습니다.

상대 경로 사용

모든 링크와 에셋은 상대 경로라서 repository Pages 경로에서도 그대로 열립니다.

기능 범위

로그인, 저장, 업로드, API 호출은 제거했고 화면 미리보기와 소개만 남겼습니다.
